# 05 - 决策记录与行动模式

## 来源：用户自我描述（一手素材）

### 重要决策方法论

**决策链路**：
AI咨询 → 认识AI的局限性（会迎合）→ 网络搜索（国内外）→ 关注国外大佬见解 → 统计汇总 → 制定计划 → 执行

这个链路体现了：不盲信任何单一信息源，有系统性的交叉验证意识。

**技术决策**：
读技术文档 → Quick Start → 快速实践出效果 → 给自己反馈 → 不断迭代
核心逻辑：实践出反馈，反馈驱动迭代。不陷入理论分析瘫痪。

**问题解决**：
第一性原理 → 底层一步步拆解 → debug式排查
核心逻辑：不猜测，从根因开始推。

### 关键决策模式

**谨慎与果决的张力**：
- 重要决定：非常谨慎 → 多方验证 → 考虑很多东西 → 制作计划
- 日常执行：列出清单 → 一一实现 → 不拖泥带水

**信息源优先级**：
国外技术大佬 > AI建议 > 国内普通资源
说明：有明确的品味判断和信源筛选意识
